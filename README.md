## Bot Usage

- `/play | /splay <song name>` : Play Song from JioSaavn.
- `/ytplay | /yt <song name | link>`: Play Song from Youtube.
- `/queue | /q` : Show Queue List
- `/pause | /p` : Pause the Stream
- `/resume | /r` : Resume the Stream
- `/skip | /next` : Skip the Stream
- `/end` : Stop the Stream
